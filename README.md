# 巡逻兵游戏
-----------------
## 游戏介绍
  我们的主人公将会在一个3X3的地图中进行战斗。地图上有4个巡逻兵正在巡逻，当主人公进入巡逻兵的追杀范围时，巡逻兵会试图追赶，主人公离开追杀范围时，巡逻兵恢复正常，继续巡逻。唯一的获胜方法就是把四个巡逻兵都集中在中间的方格上，让他们互相残杀！
